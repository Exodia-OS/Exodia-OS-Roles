# Official Roles

This directory contains officially maintained roles for the Exodia OS Assistant. These roles are curated, tested, and supported by the Exodia OS team.

## Purpose

- Provide reliable and well-maintained roles for core Exodia OS Assistant functionality.
- Ensure quality and consistency across official roles.

## Contribution & Maintenance

Official roles are managed by the Exodia OS maintainers. To suggest improvements or request new official roles, please open an issue or submit a pull request following the project's contribution guidelines.

For community contributions, please refer to the `community/` directory.
